Seguridad

En ADAE estamos conscientes de que la seguridad es un tema importante
y por defecto al crear una nueva cuenta en esta plataforma el
método de seguridad por defecto son códigos por correo, pero
puedes agregar más opciones de seguridad.

Aplicación de autenticación

Descarga una aplicación en tu teléfono para generar códigos de seguridad únicos. Las aplicaciones sugeridas incluyen Google Authenticator.

Correo (por defecto)

Por defecto se le enviará un correo a su correo institucional para verificar su identidad y las próximas veces que inicie sesión en un dispositivo nuevo se le enviará un código de verificación ahí.
